#  Retail Sales Data with Seasonal Trends and Marketing

## 📌 Project Overview
This project focuses on analyzing and predicting retail sales using a real-world dataset containing product, supplier, and sales information. The goal is to understand key factors affecting sales and build a predictive model.

---

## 📂 Dataset
Dataset Name: Retail Sales Data with Seasonal Trends and Marketing  
Source: Kaggle  
Link: https://www.kaggle.com/datasets/abdullah0a/retail-sales-data-with-seasonal-trends-and-marketing  

### Features:
- YEAR
- MONTH
- SUPPLIER
- ITEM CODE
- ITEM DESCRIPTION
- ITEM TYPE
- RETAIL SALES
- RETAIL TRANSFERS
- WAREHOUSE SALES

---

## 🎯 Objective
To predict **Retail Sales** using available features and analyze patterns affecting sales performance.

---

## ❓ Research Questions
1. How do month and seasonal trends affect retail sales?
2. Which item types generate the highest sales?
3. Does supplier influence sales performance?
4. What is the relationship between retail sales and warehouse sales?
5. Can retail sales be predicted using basic features?
6. How accurate is a simple regression model?
7. What are the key drivers of sales?

---

## ⚙️ Methodology
1. Data Loading
2. Data Cleaning (handling missing values)
3. Feature Selection
4. Encoding categorical variables
5. Train-test split
6. Model training (Linear Regression)
7. Evaluation (MAE, RMSE, R²)
8. Visualization

---

## 🤖 Model Used
- Linear Regression

---

## 📊 Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## 📈 Results
- MAE: ~8.13  
- RMSE: ~26.19  
- R² Score: ~0.11
